Build a production-ready full-stack web application named:

Customer Intake – Cartridge & Ink Shop Management System

OBJECTIVE

Create a real shop-counter intake system used daily to record customer service requests such as printer service, toner refill, laptop repair, photocopy services, and scanning.

The application must store data permanently in a database and allow editing later including timestamps.
